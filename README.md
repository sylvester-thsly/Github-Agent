# Project Pulse 📡🏢
> *Bridging the gap between code and communication.*

Project Pulse is an autonomous **AI Product Manager** designed to live inside your GitHub repository. It proactively monitors development activity and handles the "busy work" of professional communication.

### 🌟 Key Features
- **Daily Standups:** Automated project health summaries delivered to Microsoft Teams.
- **Human-like Interactions:** An AI persona that sounds like a teammate, not a bot.
- **Smart Scheduling:** Automaticale inly finds tim your calendar for project syncs.

### 🛠️ Technical Stack
- **Backend:** .NET 8 (C#)
- **Intelligence:** OpenRouter (GPT-4o / Claude 3.5 Sonnet)
- **Integrations:** GitHub Octokit, Microsoft Graph, Teams Webhooks
